# McCool Oilfield Services Website

Professional website for McCool Oilfield Services - Alberta-based mechanical and piping contractor since 2007.

## Features

- **5 Complete Pages**: Home, Services, About, Contact, Gallery
- **Integrated Contact Forms**: Connected to Formspree for form submissions
- **Interactive Image Gallery**: 36 project images with lightbox viewer
- **Featured Carousel**: 4 hero slides with drag/swipe support
- **Mobile Responsive**: Optimized for all devices
- **Dark Industrial Theme**: Professional oilfield aesthetic

## Technology Stack

- **React** with TypeScript
- **React Router** for client-side navigation
- **Tailwind CSS v4** for styling
- **Lucide React** for icons
- **Formspree** for form handling
